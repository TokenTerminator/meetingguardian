# meetingguardian

A windows system tray tool to check and remind you about upcoming meetings from all your outlook mailboxes

## Project Details
- **Project Type**: C# / .NET Project
- **Uploaded via**: [Code2Hub](https://github.com/Brito/Code2Hub)
- **Status**: Initial Upload

## Build and Run Instructions
This is a .NET / C# project. Restore dependencies and run using:
```bash
dotnet restore
dotnet run
```
